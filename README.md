# 姻缘簿：月老牵线局

黑客松互动作品。用户扮演月老，接收名著人物的情感委托，在候选池中选择撮合对象，并通过三回合相亲安排推动两位角色形成不同关系。

第一阶段完整样板线：

```text
求助者：林黛玉
候选对象：孙悟空
流程：接案台 -> 委托详情 -> 候选池 -> 三回合相亲 -> 姻缘簿结果记录
```

## 交付方向

- 本地互动空间版：离线可体验，入口 `index.html`，8MB 内，使用预写分支、规则数值和模板结果。
- AI API 完整版：可联网展示 LLM 实时生成对话、内心 OS 和结尾记录，但核心数值与结局仍由规则引擎控制。

## 关键文档

- `AGENTS.md`：多人 AI 协作开发规范。
- `Agent.md`：黑客松项目背景与硬约束。
- `月老牵线_PRD_研发版.md`：第一阶段研发边界、数据协议和玩法规则。
- `月老牵线_PRD_产品版.md`：产品阐述、用户体验与传播点。
- `赛题/赛题说明.md`：赛题原始说明。

## 当前原则

- 第一阶段只做“林黛玉 × 孙悟空”完整闭环。
- 本地版不依赖网络、API、CDN、远程字体或重素材。
- AI 生成只负责表达内容，不决定核心数值和结局档次。
- 多个 AI 协作前必须先阅读 `AGENTS.md`。
